== Description ==

This plugin adds dynamic data tag that gets the category image

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/categories-images-for-elementor` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the 'Posts' section and add/edit posts to see the new custom fields.
